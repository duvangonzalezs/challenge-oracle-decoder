# challenge-oracle-decoder

Reto del programa Alura Latam -ONE(Oracle Next Education) donde se propone realizar un encriptador y desencriptador de texto. El mensaje encriptado/Desencriptado se puede ingresar en mayúsculas pero este convertira esos caracteres a minúsculas, el botón copiar permite copiar el mensaje que ha sido encriptado o desencriptado!
